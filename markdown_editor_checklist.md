- [x] Hive 本地持久化 (3 个 Box: documents / settings / versions)
- [x] 文件夹管理 (卡片展示、创建/重命名/删除)
- [x] 标签系统 (FilterChip 筛选、添加/移除/新建)

### 导出
- [x] HTML 导出 (复制到剪贴板 / 下载文件)
- [x] PDF 导出 (系统打印对话框)

### 标签页系统
- [x] 多标签页栏 (水平滚动、标题显示)
- [x] 标签切换 (点击 / Ctrl+Tab / Ctrl+Shift+Tab)
- [x] 标签关闭 (× 按钮 / Ctrl+W / 中键点击)
- [x] 右键菜单 (关闭 / 关闭其他 / 关闭全部)
- [x] 新建标签 (+ 按钮)
- [x] 切换标签时自动保存

### 导航
- [x] 文档大纲面板 (H1-H6 层级，点击跳转)

### 用户体验
- [x] 深色/浅色/跟随系统主题
- [x] 自动保存 (5s/15s/30s/60s 可配置)
- [x] 键盘快捷键 (10+ 组合键)
- [x] 响应式布局 (桌面双栏 / 移动端单栏 + 切换条)
